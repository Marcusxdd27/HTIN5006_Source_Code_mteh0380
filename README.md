# Diabetes ML Classification - Source Code

## Project
Machine Learning Study on Type 2 Diabetes Classification

## Author
Marcus Teh Wei Jie

## Requirements
R 4.5.2 with the following packages:
- tidyverse
- tidymodels
- themis
- bonsai
- doParallel
- scales
- shapviz
- corrplot

## Dataset
Download `diabetes_dataset.csv` from Kaggle and place it in the same directory as the R scripts.

## How to Run
Execute scripts in order:
1. day1_data_integrity.R
2. day2_eda.R
3. day3_preprocessing.R
4. day4_model_training.R
5. day5_tuning.R
6. day6_evaluation.R
